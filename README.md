# Vue Sneakers

Учебный проект — интернет-магазин кроссовок на Vue 3.

Сайт позволяет просматривать каталог кроссовок, искать и сортировать товары, добавлять в корзину и избранное, оформлять заказы. Все данные хранятся локально, внешние серверы не используются.

Стек: Vue 3, Vite, Tailwind CSS, Vue Router.

Демо: https://igor1ii1.github.io/vue-sneakers/


## Структура репозитория

- **vue-sneakers/** — исходный код проекта (компоненты, страницы, стили, конфиг сборки)
- **task/** — документация и анализ проекта (ТЗ, оценка, оптимизация, gitflow, PR-стратегия, план развития)


## Запуск

```bash
cd vue-sneakers
npm install
npm run dev
```

Откроется на http://localhost:5173
