# Vue Sneakers

Учебный проект — интернет-магазин кроссовок на Vue 3.

Сайт позволяет просматривать каталог кроссовок, искать и сортировать товары, добавлять в корзину и избранное, оформлять заказы. Все данные хранятся локально, внешние серверы не используются.

Стек: Vue 3, Vite, Tailwind CSS, Vue Router.


## Структура репозитория

```
├── vue-sneakers/       — исходный код проекта
│   ├── src/            — компоненты, страницы, стили
│   ├── public/         — картинки, иконки, данные товаров (items.json)
│   ├── vite.config.js  — конфигурация сборки
│   └── package.json    — зависимости
│
└── task/               — документация и анализ проекта
    ├── task.md         — формулировка задачи (аудитория, трафик, продукт)
    ├── terms.md        — техническое задание (стек, userflow, компоненты)
    ├── review.md       — оценка проекта и сравнение с конкурентами
    ├── optimize.md     — план оптимизации (что и как улучшать)
    ├── grow.md         — план развития (roadmap на 3–12 месяцев)
    ├── remark.md       — обсуждение улучшений
    ├── gitflow.md      — стратегия ветвления и правила коммитов
    ├── prstrategy.md   — стратегия продвижения
    └── readme.md       — подробное описание проекта
```


## Запуск

```bash
cd vue-sneakers
npm install
npm run dev
```

Откроется на http://localhost:5173
