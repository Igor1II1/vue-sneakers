# Техническое задание — Vue Sneakers

## 1. Общее описание

Vue Sneakers — интернет-магазин кроссовок, реализованный как одностраничное приложение (SPA) на Vue 3. Проект позволяет просматривать каталог кроссовок, искать и сортировать товары, добавлять их в корзину и избранное, а также оформлять заказы.

Все данные хранятся на стороне клиента (localStorage + статический JSON). Внешний бэкенд не требуется.


## 2. Технологический стек

- **Фреймворк:** Vue 3 (Composition API, `<script setup>`), версия 3.5+
- **Сборщик:** Vite, версия 6.x
- **Стили:** Tailwind CSS, версия 4.x
- **Маршрутизация:** Vue Router, версия 4.x
- **Анимации:** @formkit/auto-animate, версия 0.9+
- **Утилиты:** lodash.debounce, версия 4.x
- **Линтинг:** ESLint + Prettier + Oxlint


## 3. UserFlow (пользовательские сценарии)

### Сценарий 1: Просмотр каталога
1. Пользователь открывает главную страницу
2. Загружается каталог из 12 кроссовок (из локального JSON)
3. Отображается сетка карточек (4 колонки) с фото, названием и ценой
4. Пользователь может отсортировать товары: по названию, по цене (дешёвые), по цене (дорогие)
5. Пользователь может ввести запрос в поле поиска — каталог фильтруется в реальном времени (с debounce 300ms)

### Сценарий 2: Добавление в корзину
1. На карточке товара пользователь нажимает кнопку «+»
2. Товар добавляется в массив корзины, иконка меняется на галочку
3. В шапке обновляется общая сумма
4. Корзина автоматически сохраняется в localStorage
5. При повторном нажатии товар удаляется из корзины

### Сценарий 3: Работа с корзиной (Drawer)
1. Пользователь нажимает на иконку корзины в шапке
2. Справа выезжает панель (Drawer) с затемнением фона
3. Отображается список товаров в корзине с ценами
4. Показывается итого и налог 5%
5. Каждый товар можно удалить из корзины крестиком
6. Если корзина пуста — показывается заглушка «Корзина пустая»
7. Кнопка «Оформить заказ» сохраняет заказ в localStorage и показывает подтверждение

### Сценарий 4: Избранное (закладки)
1. На карточке товара пользователь нажимает иконку сердца
2. Товар помечается как избранный, иконка становится красной
3. Данные сохраняются в localStorage
4. На странице «Закладки» (/favorites) отображаются все избранные товары
5. Повторное нажатие на сердце убирает товар из избранного

### Сценарий 5: Навигация
1. Клик на логотип — переход на главную
2. Клик на «Закладки» — переход на страницу избранного
3. Переходы без перезагрузки страницы (SPA)


## 4. Структура компонентов

- **App.vue** — корневой компонент, управляет корзиной
  - **Header.vue** — шапка с логотипом, суммой корзины, навигацией
  - **Drawer.vue** — боковая панель корзины
    - **DrawerHead.vue** — заголовок корзины с кнопкой закрытия
    - **CartItemList.vue** — список товаров в корзине
      - **CartItem.vue** — один товар в корзине
    - **InfoBlock.vue** — заглушки (корзина пуста / заказ оформлен)
  - **router-view** — переключение страниц
    - **Home.vue** — главная страница с каталогом (CardList > Card)
    - **Favorites.vue** — страница избранного (CardList)


## 5. Хранение данных

- **Каталог товаров:** `/public/items.json` — Статический файл с 12 товарами
- **Корзина:** `localStorage('cart')` — Массив товаров, добавленных в корзину
- **Избранное:** `localStorage('favorites')` — Массив объектов `{id, item_id}`
- **Заказы:** `localStorage('orders')` — Массив оформленных заказов


## 6. Маршрутизация

- **`/`:** Home.vue — Главная страница с каталогом
- **`/favorites`:** Favorites.vue — Страница избранных товаров


## 7. Дизайн-решения

- Фон страницы: светло-зелёный (`rgb(247 254 231)`)
- Контейнер: белый с тенью и скруглёнными углами, ширина 80% экрана
- Карточки товаров: белый фон, рамка, скругление, тень при наведении, анимация подъёма
- Drawer: фиксированная панель справа (384px), полупрозрачный оверлей
- Шрифты и цвета: стандартные Tailwind, акцентный цвет — lime-500
- Адаптивность: сетка из 4 колонок (grid-cols-4)
