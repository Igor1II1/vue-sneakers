# План развития Vue Sneakers (Roadmap)

## 1. Краткосрочные улучшения (1–3 месяца)

### Адаптивная вёрстка
Переделать сетку карточек под мобильные устройства. На телефонах — 1 колонка, на планшетах — 2, на десктопе — 4. Адаптировать шапку, Drawer и все всплывающие элементы.

### Расширение каталога
Увеличить количество товаров до 50–100 позиций. Добавить новые бренды: Adidas, New Balance, Asics, Converse.

### Фильтры
Добавить фильтрацию по бренду, ценовому диапазону. В перспективе — по размеру и цвету.

### Страница товара
Отдельная страница для каждого товара с увеличенным фото, описанием, выбором размера и кнопкой «В корзину».

### Бэкенд
Разработать серверную часть на Node.js (Express или Fastify) с базой данных (PostgreSQL или MongoDB). Перенести хранение товаров, заказов, избранного на сервер.

### Регистрация и авторизация
Реализовать вход и регистрацию через email/пароль. Подключить JWT-токены для сессий. В будущем — вход через VK или Google.


## 2. Среднесрочные улучшения (3–6 месяцев)

### Личный кабинет
Страница профиля с историей заказов, сохранёнными адресами доставки и управлением аккаунтом.

### Оплата
Подключить платёжный агрегатор (ЮKassa или CloudPayments) для приёма оплаты картой и через СБП.

### Доставка
Интеграция с курьерской службой (СДЭК, Boxberry). Расчёт стоимости доставки по адресу. Отслеживание статуса заказа.

### Отзывы и рейтинги
Дать возможность покупателям оставлять отзывы и ставить оценки кроссовкам. Показывать средний рейтинг на карточке товара.

### Уведомления
Email-уведомления о статусе заказа. Push-уведомления о скидках и новых поступлениях.

### Поиск
Полнотекстовый поиск с подсказками. Поиск по артикулу. История поисковых запросов.


## 3. Долгосрочные улучшения (6–12 месяцев)

### Мобильное приложение
Разработка приложения на React Native или Flutter для iOS и Android с полным функционалом магазина.

### Программа лояльности
Система бонусных баллов за покупки. Накопительные скидки для постоянных покупателей. Промокоды.

### Персонализация
Рекомендации на основе истории просмотров и покупок. Раздел «Вам может понравиться» на главной странице и на странице товара.

### Админ-панель
Интерфейс для управления каталогом, заказами, пользователями и акциями без необходимости менять код.

### Аналитика
Подключение Яндекс.Метрики. Отслеживание воронки: просмотр → корзина → заказ → оплата. A/B тестирование элементов интерфейса.

### Мультиязычность
Английская версия сайта для выхода на рынки СНГ и за рубеж.
