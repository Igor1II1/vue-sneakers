# Формулировка задачи — Vue Sneakers

## 1. На какую аудиторию рассчитываем

Целевая аудитория — молодые люди от 16 до 30 лет, которые интересуются кроссовками и уличной модой. Это студенты, молодые специалисты и все, кто следит за релизами Nike, Puma, Jordan, Reebok и других брендов.

### Сегмент A: Студенты (16–22 года)

Характеристики:
- Активно следят за трендами уличной моды
- Выбирают кроссовки не только по удобству, а по стилю
- Часто сравнивают цены в нескольких магазинах перед покупкой
- Покупают 1–2 пары в сезон, предпочитают акции и скидки
- Основной канал — мобильные устройства

Проблемы при покупке:
- Крупные магазины перегружены товарами, сложно найти нужную модель
- Цены завышены из-за накруток посредников
- Интерфейсы маркетплейсов неудобные, много лишнего

### Сегмент B: Молодые специалисты (23–30 лет)

Характеристики:
- Готовы платить за качество и бренд
- Ценят удобство покупки: быстрый поиск, понятная навигация
- Покупают кроссовки для разных задач: спорт, повседневная носка, стиль
- Используют и десктоп, и мобильные устройства

Проблемы при покупке:
- Слишком много времени уходит на поиск нужной модели
- Нет удобных фильтров по цене и названию в одном месте
- Хочется простой интерфейс без отвлекающих баннеров и рекламы

### География

Основной рынок — Россия и СНГ. Интерфейс на русском языке. Цены в рублях.


## 2. На какой трафик рассчитываем

Проект учебный, поэтому цифры трафика ориентировочные, но заложены с расчётом на реальный запуск.

- Первый месяц: 300–500 уникальных посетителей (друзья, однокурсники, публикация в соцсетях)
- Через 3 месяца: 1000–2000 уникальных посетителей в месяц (SEO, контент в VK и Telegram)
- Через 6 месяцев: 3000–5000 уникальных посетителей в месяц (при активном продвижении)

Основные источники трафика:
- Прямые заходы (ссылка из портфолио, резюме)
- Социальные сети (VK, Telegram)
- Поисковые системы (Яндекс, Google) при базовой SEO-оптимизации


## 3. Что хотим продать

Основной продукт — кроссовки известных брендов:
- Nike (Blazer Mid, Air Max 270, Kyrie, LeBron, Jordan)
- Puma (Future Rider, Suede Classic)
- Under Armour (Curry 8)
- Reebok (Club C 85)

Ценовой диапазон: от 7 900 до 16 400 руб.

Средний чек: ~12 000 руб.

В каталоге 12 позиций — достаточно для демонстрации работы магазина. При масштабировании каталог можно расширить до сотен позиций.


## 4. Как мы этого будем добиваться

Стратегия строится на трёх вещах: простота, скорость, удобство.

### Продуктовые решения:
- Минималистичный интерфейс без лишнего — только товары, поиск и корзина
- Мгновенная сортировка (по цене, по названию) и поиск по каталогу
- Корзина с подсчётом итоговой суммы и налога
- Система закладок (избранное) для сохранения понравившихся моделей
- Все данные хранятся локально — сайт работает без внешних серверов

### Технические решения:
- Vue 3 с Composition API — современный и быстрый фреймворк
- Tailwind CSS — адаптивный дизайн, быстрая вёрстка
- Vite — мгновенная сборка и горячая перезагрузка при разработке
- Данные товаров в статическом JSON — нет зависимости от внешних API
- localStorage для корзины, избранного и заказов — всё работает офлайн

### Маркетинговые решения:
- Размещение проекта на GitHub Pages для бесплатного хостинга
- Публикация в портфолио и на профильных площадках
- Простая ссылка, которую можно отправить кому угодно — сайт откроется и заработает
