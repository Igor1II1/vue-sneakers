# Vue Sneakers — интернет-магазин кроссовок

Vue Sneakers — это веб-приложение интернет-магазина кроссовок, построенное на Vue 3. Проект включает каталог товаров с поиском и сортировкой, корзину с подсчётом стоимости, систему избранного и оформление заказов. Все данные хранятся локально — сайт полностью автономен и не зависит от внешних серверов.


## Цели и аудитория

**Цель:** создать быстрый и удобный интернет-магазин кроссовок с минимальным интерфейсом, без лишних элементов, с фокусом на каталог и процесс покупки.

**Целевая аудитория:** молодые люди 16–30 лет, которые интересуются кроссовками Nike, Puma, Jordan, Reebok, Under Armour. Ценят простой интерфейс и быструю работу сайта.


## Основные функции

- Каталог из 12 моделей кроссовок с фотографиями, названиями и ценами
- Сортировка товаров: по названию, по цене (дешёвые), по цене (дорогие)
- Поиск по названию с задержкой ввода (debounce)
- Корзина с боковой панелью (Drawer): добавление, удаление товаров, подсчёт итого и налога 5%
- Избранное (закладки): добавление товаров в избранное, отдельная страница со списком
- Оформление заказа с сохранением в localStorage
- Сохранение корзины между сессиями (localStorage)
- Анимации добавления/удаления товаров (auto-animate)
- Навигация между страницами без перезагрузки (SPA)


## Технологический стек

- **Vue 3 (Composition API):** Фреймворк, `<script setup>`, ref, computed, watch, provide/inject
- **Vite:** Сборка и dev-сервер
- **Tailwind CSS 4:** Стилизация компонентов
- **Vue Router 4:** Маршрутизация (главная, избранное)
- **@formkit/auto-animate:** Плавные анимации списков
- **lodash.debounce:** Задержка поиска при вводе
- **ESLint + Prettier + Oxlint:** Линтинг и форматирование кода


## Структура проекта

- **public/** — картинки кроссовок (12 шт.), иконки, логотип, данные товаров (items.json)
- **src/main.js** — точка входа, настройка роутера и плагинов
- **src/App.vue** — корневой компонент, логика корзины
- **src/assets/main.css** — глобальные стили и подключение Tailwind
- **src/components/** — Header, Card, CardList, Drawer, DrawerHead, CartItem, CartItemList, InfoBlock
- **src/pages/** — Home.vue (каталог), Favorites.vue (избранное)


## Запуск проекта

### Установка и запуск

```bash
git clone <url-репозитория>
cd vue-sneakers
npm install
npm run dev
```

После запуска откройте в браузере: http://localhost:5173

### Сборка для продакшена

```bash
npm run build
npm run preview
```

### Деплой на GitHub Pages

```bash
npm run deploy
```


## Примечания для разработки

- Все изображения кроссовок хранятся локально в папке `public/sneakers/`
- Данные товаров — в файле `public/items.json`, при необходимости можно добавлять новые позиции
- Корзина, избранное и заказы сохраняются в localStorage браузера
- Для подключения бэкенда нужно заменить fetch-запросы к JSON на запросы к API
- Рекомендуемый редактор: VS Code с расширением Volar
